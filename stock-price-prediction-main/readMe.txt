# 📈 Stock Price Predictor

> A Streamlit web app to forecast stock trends using **LSTM** and **Random Forest**, with secure login, admin control, and saved model management.

## 🚀 Features

- 🔑 Secure Login & Signup
- 👑 Admin Dashboard: manage users, logs, models
- 📉 Predict with:
  - LSTM (Deep Learning)
  - Random Forest (ML Ensemble)
- 🧠 Hyperparameter tuning
- 💾 Save, rename, delete, load models
- 📊 Charts for:
  - Predictions vs Actual
  - 30-day forecast
- 📈 Yahoo Finance integration (real-time prices)
