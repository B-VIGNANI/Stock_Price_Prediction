How to set the website up

1. Ensure Python 3.10 is installed.

2. Set up a virtual environment
    a. Open terminal
    b. Enter -- python -m venv venv
    c. Enter -- venv\Scripts\activate

3. Install dependencies with -- pip install -r requirements.txt

4. Enter -- streamlit run app.py

5. Set up the app - Create an account under the register tab and login using your new credentials.
    a. You have the option to change your password.

6. In the price predictor tab:
    a. Enter a stock ticker (eg. TSLA)
    b. Choose data range (via slider)
    c. Click "Train Model & Predict"
    d. See visualisations and future forecasts
    e. Click "Save Model" to store your Model

7. In the Saved Models tab:
    a. View, rename, delete or reload past Models
    b. Reloaded models instantly show saved predictions.
